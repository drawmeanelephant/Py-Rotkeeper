📖 Welcome to Rotkeeper

Rotkeeper is a Python-based static site generator that turns Markdown into HTML using Pandoc.
Every page has a purpose — understanding the structure keeps your builds neat and predictable.

Key Sections

Here are the major areas of the documentation:

  • Quickstart – Install, initialize, and run your first build
  • Pipeline – Core commands: render, assets, collect_assets
  • Sitemap – How navigation and site maps are generated
  • Scaffold – init and reseed project setup
  • Utilities – Helpful commands like nav, cleanup_bones, book
  • Concepts – How Rotkeeper reads frontmatter
