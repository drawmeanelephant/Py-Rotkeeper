pipeline

  • rc assets Catalogs static assets from bones/assets/ and page-local files, writing assets.yaml for use by collect_assets.
  • rc collect-assets Copies cataloged assets from source into output/, using SHA-256 hashes to skip unchanged files.
  • Pipeline Overview of the Rotkeeper build pipeline: render, assets, and collect_assets.
  • rc render Compiles Markdown to HTML via Pandoc, applies templates, and compiles SCSS.
  • rc sitemap-pipeline Full sitemap pipeline: collects pages, builds metadata trees, writes index pages, nav partial, sidecar metadata, and a unified YAML report.
