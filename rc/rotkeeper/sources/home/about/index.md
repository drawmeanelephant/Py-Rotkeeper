# About Rotkeeper

Rotkeeper started as a Bash project, became a Python CLI, and is still
getting weirder. It uses Pandoc to render Markdown to HTML and is designed
to work across multiple sites by running from each site's directory.

**Project:** [Py-Rotkeeper on GitHub](https://github.com/drawmeanelephant/Py-Rotkeeper)  
**Author:** Filed Systems
