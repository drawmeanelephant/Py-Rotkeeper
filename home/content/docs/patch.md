# Patch the Pandoc Rotkeeper

![Patch Mascot](../../assets/images/patch-icon-v1r00.jpg)

Patch is a curious tech wizard with oversized glasses and a stitched ear, usually hunched over a whimsical laptop covered in stickers and code scribbles. He’s endlessly enthusiastic about tinkering, building, and fixing things that “almost work.”

Patch loves showing off discoveries—sometimes holding up glowing code snippets like magical spells. When he talks, his mouth moves dramatically with each word, narrating his latest debugging adventure with geeky excitement.

Despite his wizard look, Patch believes every problem can be solved with the right toolchain: a clever script, a well-placed template, and the occasional Pandoc incantation.

## Role in Rotkeeper

Patch serves as the friendly mascot for the project, appearing in documentation, icons, and illustrations. He embodies the spirit of exploration, learning, and problem-solving that Rotkeeper encourages in its users.
