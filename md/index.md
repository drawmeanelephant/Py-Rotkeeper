# 📖 Welcome to Rotkeeper

Rotkeeper is a Python-based static site generator that turns Markdown into HTML using [Pandoc](https://pandoc.org).  
Every page has a purpose — understanding the structure keeps your builds neat and predictable.

---

## Key Sections

Here are the major areas of the documentation:

- [Quickstart](quickstart.html) – Install, initialize, and run your first build  
- [Pipeline](pipeline/index.html) – Core commands: `render`, `assets`, `collect_assets`  
- [Sitemap](sitemap/index.html) – How navigation and site maps are generated  
- [Scaffold](scaffold/index.html) – `init` and `reseed` project setup  
- [Utilities](utils/index.html) – Helpful commands like `nav`, `cleanup_bones`, `book`  
- [Concepts](concepts/pandoc-frontmatter.html) – How Rotkeeper reads frontmatter
